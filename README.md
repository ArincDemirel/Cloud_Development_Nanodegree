# Cloud Development Nanodegree

This repository contains projects from the Cloud Development Nanodegree program.

## Projects

### cd12099-Full-Stack-Apps-AWS

A Node.js image filtering service deployed on AWS Elastic Beanstalk. This project processes images by converting them to greyscale and resizing them to 256x256 pixels.

**Location:** [Cloud_Development_Nanodegree/cd12099-Full-Stack-Apps-AWS/](Cloud_Development_Nanodegree/cd12099-Full-Stack-Apps-AWS/)

**Features:**
- Image filtering API endpoint
- S3 integration for image storage
- AWS Elastic Beanstalk deployment
- RESTful API with error handling

See the project README for detailed documentation: [Cloud_Development_Nanodegree/cd12099-Full-Stack-Apps-AWS/README.md](Cloud_Development_Nanodegree/cd12099-Full-Stack-Apps-AWS/README.md)
